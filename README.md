## Dever de casa

Antes da segunda-feira:

1. **Instale o Claude CoWork** no seu computador principal — não no que você usa para "experimentar".
2. **Refine a skill** que você construiu no bloco "Assumindo o controle". Use durante a semana. Anote onde ela falha.
3. **Identifique uma tarefa semanal recorrente** que pode virar a próxima skill. Não construa ainda. Só observe.

Em **2–4 semanas** faremos uma clínica de follow-up para revisar onde vocês chegaram.
