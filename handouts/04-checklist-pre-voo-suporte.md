## Erros comuns + como resolver

| Sintoma | Provável causa | O que fazer |
|---|---|---|
| "App não baixa" | Wifi corporativo bloqueia | Usar 4G/5G do celular como hotspot |
| "Login fica girando" | Cookie / cache | Fechar e reabrir; se persistir, login no claude.ai primeiro |
| "Pede cartão de crédito" | Trial expirou | Manos avisa qual conta usar (compartilhada do dia) |
| "Não vejo CoWork no menu" | Versão desatualizada | Atualizar app pelo App Store / Play Store |
| "Não consigo arrastar arquivo" | Permissão de Files (macOS) / Storage (Windows) | Settings do SO → app permissions → liberar |
| "Connector Google não autoriza" | Pop-up bloqueado no browser | Liberar pop-up para `claude.ai` no browser |
| "Linux user" | App nativo é Mac/Win | Versão web em `claude.ai` é OK pra hoje, perdem só computer use |
